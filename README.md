# Delicious — Restaurant Website

Un site web moderne et responsive inspiré du template **Delicious** de BootstrapMade.  
Le projet est développé en **HTML, CSS/SASS/SCSS, Bootstrap 5**.

---

## Description

Ce site représente un restaurant avec un design professionnel et élégant.  
Il inclut plusieurs sections essentielles comme :

- **Hero** — image principale avec texte et bouton.
- **About** — présentation du restaurant.
- **Why Us** — arguments et points forts.
- **Menu** — catégories et plats du restaurant.
- **Specials** — plats spéciaux.
- **Events** — évènements & offres.
- **Chefs** — équipe des chefs avec effet hover et icônes sociales.
- **Gallery** — galerie d’images responsive.
- **Contact** — informations, Google map et formulaire.
- **Footer** — liens rapides & réseaux sociaux.

Toutes les sections sont 100% responsive.

## Fonts & Colors

### Palette de couleurs utilisée

 **Primary**   `#cda45e`  Boutons, titres, icônes actives 
 **Dark**      `#0c0b0b`  Texte foncé, navbar, footer 
 **Light**     `#ffffff`  Background clair, texte inversé 
 **Gray**      `#444444`  Sous-titres, paragraphes 
**Background**  `#1a1818` Sections sombres (footer + NavFix) 
**Button&Special**  `##ffb03b`


Le site utilise :

- **Font principale :** `Open Sans`, sans-serif  
  → utilisée pour les paragraphes et textes généraux

- **Titres / Headings :** `'Satisfy', sans-serif`, serif  
  → pour donner un style élégant et restaurant haut de gamme
